Cauce
Dibujás el proceso. El sistema lo ejecuta.
Un editor visual para procesos de hospitales y trámites del Estado — donde el diagrama se convierte, solo, en las pantallas que opera el personal.
EL PRINCIPIO
Una sola definición,
dos mundos.
Lo que diseña el configurador es una plantilla. Lo que corre con datos reales es un caso. El motor lee la misma definición para dibujar el diagrama y para renderizar las pantallas. Nadie programa una pantalla a mano.
